Representantes

Incluir representantes a la empresa

Para agregar los representantes de la Empresa, acceder a la solapa Representantes.

Al insertar un nuevo Representante, se abre la ventana Seleccion de cargo fiscal, para introducir el CIF / DNI.

Si el CIF/DNI no existe, se presenta la Lista de Referencia para dar de alta un nuevo registro.

Una vez introducidos los datos en la anterior ventana, el Representante ya esta creado y asociado a la empresa.
